# Obsidian 入门指南

## 一、Obsidian 是什么？

Obsidian 是一款强大的知识管理和笔记软件，核心特点是：
- **本地存储**：所有笔记都是 Markdown 文件，存储在你的电脑上
- **双向链接**：笔记之间可以相互链接，形成知识网络
- **图谱视图**：可视化展示笔记之间的关系

---

## 二、基本操作

### 1. 创建笔记
- 快捷键：`Ctrl + N` 创建新笔记
- 或点击左侧文件列表的"新建笔记"按钮

### 2. Markdown 基础语法

```markdown
# 一级标题
## 二级标题
### 三级标题

**粗体文字**
*斜体文字*
~~删除线~~

- 无序列表项 1
- 无序列表项 2

1. 有序列表项 1
2. 有序列表项 2

> 引用文字

`行内代码`

[链接文字](链接地址)
```

### 3. 创建链接（核心功能）

```
[[笔记名称]]           # 链接到其他笔记
[[笔记名称|显示文字]]   # 自定义显示文字
[[笔记名称#标题]]      # 链接到笔记中的某个标题
```

**示例**：
```
查看 [[超市业务]] 了解更多
```

### 4. 标签系统
```
#工作 #重要 #待办
```

## 四、Trae + Obsidian 联动方案

### 已配置：软链接
已在 `d:\AI编程\trae\obsidian` 创建了指向 Obsidian 的软链接，Trae 可以直接读写 Obsidian 笔记。

### 推荐工作流程

```
┌─────────────────────────────────────────────────────────┐
│                    Trae + Obsidian 工作流                │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  1. 需求阶段                                            │
│     Obsidian 写需求文档 → Trae 读取并理解需求            │
│                                                         │
│  2. 开发阶段                                            │
│     Trae 生成开发计划 → 保存到 Obsidian                  │
│     Trae 编写代码 → 记录技术决策到 Obsidian              │
│                                                         │
│  3. 总结阶段                                            │
│     Trae 生成项目文档 → 整理到 Obsidian                  │
│     在 Obsidian 中建立知识链接                           │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 实用场景

1. **需求文档**：在 Obsidian 中编写需求，让 Trae 读取
2. **开发计划**：Trae 生成的计划保存到 Obsidian
3. **API 文档**：Trae 生成后保存到 Obsidian
4. **问题记录**：遇到问题时记录到 Obsidian，方便 Trae 参考
5. **知识沉淀**：项目完成后整理经验到 Obsidian
